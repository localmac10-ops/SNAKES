# Contributing to SNAKES

We welcome contributions from everyone! Please read these guidelines to get started:

## How to Contribute
- Fork the project and clone your fork
- Create a feature or fix branch (`git checkout -b feature-name`)
- Make your changes, add tests if applicable
- Commit your changes and push to your fork
- Open a pull request on GitHub, describing your change

## Code of Conduct
Please review our [Code of Conduct](CODE_OF_CONDUCT.md) to foster a welcoming community.

## Reporting Issues
- Search the [issues](../../issues) to see if your problem is already reported
- If not, open a new issue with details and steps to reproduce

## Feature Requests
- Open an issue describing your idea and its benefits

Thanks for helping make SNAKES better!
